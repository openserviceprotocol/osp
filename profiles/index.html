OSP Profile Registry

Machine-readable registry of Open Service Protocol domain profiles — branch-specific vocabularies that let AI agents compare providers within an industry.

This subdomain serves the authoritative JSON Schemas and the registry index. For human browsing, search, and documentation, see www.openserviceprotocol.org/profiles.

Machine endpoints

  • /registry/index.json — registry index (JSON)
  • /registry/index.schema.json — schema for the index
  • /profiles.txt — llms.txt-style summary for LLM discovery

Pilot profiles (v1)

  • osp-logistics-general — freight and shipping
  • osp-cold-chain-logistics — temperature-controlled transport
  • osp-consulting — strategy and advisory services
  • osp-it-managed-services — managed hosting, databases, SRE
  • osp-legal-advisory — legal services
  • osp-energy-supply — energy retail
  • osp-cleaning-services — facility cleaning

Specification

  • Profile specification — mechanism, agent behaviour, versioning rules
  • Source repository — schemas, examples, validator
Profiles are versioned under stable URLs. A profile schema URL, once published, remains byte-stable indefinitely; breaking changes ship as a new major (/v2.schema.json).
