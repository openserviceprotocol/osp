# CloudOps GmbH

> Managed cloud infrastructure and DevOps services for SMEs in 
> the DACH region. AWS and Azure certified. From single-server 
> hosting to full platform engineering with 24/7 monitoring.

## Available Services

- [Managed Hosting](/osp/services/hosting.yaml): VM and container hosting on AWS/Azure, monitored, patched, backed up
- [DevOps-as-a-Service](/osp/services/devops.yaml): CI/CD pipeline setup, infrastructure-as-code, deployment automation
- [Cloud Migration](/osp/services/migration.yaml): Assessment, planning, and execution of on-premise to cloud migration
- [24/7 Monitoring & Incident Response](/osp/services/monitoring.yaml): Proactive monitoring, alerting, first-response for production systems
- [Security Audit](/osp/services/security-audit.yaml): Infrastructure security assessment, penetration testing, compliance review

## Not Available

- Software development (we operate infrastructure, we don't build applications)
- Google Cloud Platform (AWS and Azure only)
- On-premise hardware procurement or installation
- Data center colocation

## Conditions

- Minimum contract for managed services: 12 months
- One-time services (migration, audit): no minimum term
- Regions: DACH (infrastructure can be global)
- Languages: de, en
- Incident response time: < 15 min (24/7 contracts), < 1h (business hours contracts)
